# 01 — Бриф проекта

## Что продаём

Квартира в Абовяне (пригород Еревана), прямая продажа от собственника.  
Не агентство. Не посредник. Один объект — единственный в этом доме на продажу.

## Целевая аудитория (приоритет)

### Первичная: армяне диаспоры
- Живут за рубежом (США, Россия, Франция, Германия, Канада, ОАЭ)
- Хотят «связь с Родиной»: дом для отдыха, для родителей, для будущего переезда
- Покупают удалённо — нужен доверительный, визуальный, простой лендинг
- Ценят прямую сделку без комиссий агентов (экономия 3–5%)
- Языки: **армянский** (основной), русский, английский

### Вторичная: локальные покупатели и инвесторы
- Ищут недвижимость с потенциалом роста в Котайке
- Знают про статую Христа на горе Атис и туристический бум Абовяна

## Главное обещание (Value Proposition)

> **Квартира с видом на будущее** — в городе, который станет туристическим центром Армении, по цене пригорода сегодня.

## Уникальные преимущества (USP)

1. **Локация у горы Атис** — строится монумент Христа (~77 м), аналог Рио
2. **Прямая продажа** — без комиссий агентов, живые фото, быстрые переговоры
3. **Единственный объект** в доме на продажу — нет конкуренции в подъезде
4. **4-й этаж** — вид на зелень и холм, не первый этаж
5. **78 м², 3+1** — просторная советская планировка с потенциалом ремонта
6. **10–15 мин до Еревана** — горный воздух + столица рядом

## Что НЕ продаём (честность)

- Интерьер **без ремонта** — советская отделка, нужна модернизация
- Это **не** «заходи и живи», это **инвестиция + потенциал**
- Фото честные, без фотошопа — это преимущество для доверия диаспоры

## Структура лендинга (формула продающего скролла)

```
1. Hero          — эмоция + вид на Атис + CTA WhatsApp
2. Боли (×3)     — узнавание проблемы покупателя
3. Механизм      — почему большинство ошибаются (цена сегодня vs инфраструктура завтра)
4. Последствия   — что будет, если подождать
5. Решение       — эта квартира + характеристики + галерея
6. История дома  — заводская застройка 60–70-х, квартал Элар, частная собственность (см. 09-building-heritage.md)
7. Локация       — «Что рядом» (Атис, церковь, экотourism, логистика)
7. Планировка    — SVG-схема 78 м²
8. Оффер         — цена $92K, выгоды, торг
9. FAQ (короткий)— 4–5 вопросов для диаспоры
10. Footer       — WhatsApp + телефон
```
