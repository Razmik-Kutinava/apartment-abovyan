## Важные решения (зафиксированы)

- **Hero-фон:** рендер статуи на горе Атис (`hatis3-rus.jpg`) — главный эмоциональный якорь, не интерьер
- **Честность:** интерьер без ремонта — подаём как «инвестиционный потенциал + локация», не как «люкс готовый»
- **Язык по умолчанию:** армянский (`hy`) — основная аудитория диаспора
- **Единственный CTA:** WhatsApp на всех языках
- **Планировка:** SVG-заглушка на основе реконструкции по фото (до получения официального плана)
