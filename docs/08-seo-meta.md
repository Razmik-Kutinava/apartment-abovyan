## Open Graph (WhatsApp / Telegram / Facebook preview)

```html
<meta property="og:type" content="website" />
<meta property="og:title" content="[locale title]" />
<meta property="og:description" content="[locale description]" />
<meta property="og:image" content="https://DOMAIN/images/og-preview.jpg" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:url" content="https://DOMAIN/[locale]/" />
<meta property="og:locale" content="hy_AM" /> <!-- or ru_RU, en_US -->
```

### OG Image spec
- Size: 1200×630 px
- Content: hatis3-rus.jpg background + text overlay
- Text: «78 м² · Абовян · $92 000 · Прямая продажа»
- Create at build time or manually once
