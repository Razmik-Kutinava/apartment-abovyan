## Честная подача фото

> **Важно для диаспоры:** интерьер без ремонта виден на фото.  
> Не ретушировать. В галерее добавить подпись: «Реальные фото — без фильтров».  
> Акцент маркетинга — на **локацию и потенциал**, не на «luxury interior».

### Порядок в галерее (storytelling)
1. Лodжия с видом на зелень (wow-фактор)
2. Зал с окном (пространство)
3. Комната с балконом (планировка)
4. Балкон снаружи (вид с 4-го этажа)
5. Прихожая / паркет (solid bones)
6. Кухня (функциональность)
7. Остальные комнаты
8. Санузлы (честность)
