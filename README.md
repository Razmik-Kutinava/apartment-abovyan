# Salehouse — Garni 10, Abovyan

Лендинг продажи квартиры. Astro SSG, 3 языка (hy / ru / en).

## Запуск

```bash
npm install
npm run dev      # http://localhost:4321/hy/
npm run build
npm run preview
```

## Деплой (Vercel + GitHub)

Репо: [apartment-abovyan](https://github.com/Razmik-Kutinava/apartment-abovyan)

1. Push в `main` → Vercel автодеплой (после import проекта)
2. Инструкция: [docs/10-deploy-vercel.md](./docs/10-deploy-vercel.md)
3. Env на Vercel: `SITE_URL=https://your-domain.app`

## Документация

- `docs/` — спеки, контент, дизайн
- `ops/` — операционка, задачи, journal
- `.cursor/rules/task-workflow.mdc` — workflow агента
